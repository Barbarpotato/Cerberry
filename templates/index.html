Technology Stack

Explore the powerful features and technology stack that Cerberry Backend utilizes

Backend Framework

Cerberry Backend Service use the Flask Framework. Flask is a micro web framework written in Python. It is designed to be lightweight and modular, allowing developers to build web applications quickly and with minimal code.

CI/CD Pipeline

CI/CD (Continuous Integration and Continuous Deployment) is a software development practice that automates testing and code deployment continuously. Cerberry use GitHub, Vercel, and GitHub Actions

Containerized Application

Cerberry Backend use Docker to package, distribute, and run applications in an isolated environment called a " container." These containers encapsulate the software and all its dependencies, allowing applications to run consistently across various environments
